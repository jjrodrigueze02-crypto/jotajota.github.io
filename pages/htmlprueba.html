HTML, siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar para estructurar el contenido de las páginas web. Define el contenido como texto, imágenes, enlaces y otros elementos usando un sistema de etiquetas que le dicen al navegador cómo organizarlos y presentarlos al usuario. Fue creado por Tim Berners-Lee en 1991 y ha evolucionado a lo largo de los años, con la versión más reciente siendo HTML5, que incluye nuevas características para multimedia, gráficos y aplicaciones web. HTML es fundamental para el desarrollo web y se utiliza junto con CSS (Cascading Style Sheets) y JavaScript para crear sitios web interactivos y visualmente atractivos.
